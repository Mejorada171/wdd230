Visual Hierarchy

DOORDASH

doordash.com

Visual hierarchy is a methodology that organizes design elements by importance, creating a user experience (UX) that draws attention to the focal points you want users to interact with

The screenshot above (DOORDASH Web page) is an example of Visual Hierarchy as a design principle, because the background color contrast and stand out te most important text of the home page like: the "What of the page?", the searching bar, the sign in button, ths sign up button and yeah! The brand name. These elements are in text color white, some bolder, some with a special small background in every text. Here we definetly find some Visual Hierarchy elements.

White Space and Clean Design

Apple

apple.com

Whitespace (also known as negative space) is the empty space around the content and elements on a web page. It is used to improve the visual experience for the user.

The screenshot above is a great example of the use White Space and Clean Design in a Web Page, notice how Apple Web Page place all the elements with a space around and between them, every image, text, or link is apart from the other but at the same time, one characteristic from the right use of White Space is make the web page exclusive or luxury and Apple web page definitely did it.

Contrast

Bitso
bitso.com

Contrast is the practice of making design elements look different from each other in order to make the important elements stand out

The screenshot above is a clear example of a Web Page with a great contrast between their elements, when we get in the Web Page we may notice the elements and the great contrast between them, for example, using a black background color and a bold white color text we immediately look at the first promise for the company "Your safe, easy, borderless account", another great contrast element is the "Start now" button, that shiny green background color of the button and the color white text really call to make that action, accomplishing the contrast goal.
